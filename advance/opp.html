Object orianted programming langunge

Object-oriented programming is based on the concept of objects. In object-oriented programming data structures, or objects are defined, each with its own properties or attributes. Each object can also contain its own property or methods. Software is designed by using objects that interact with one another.

Class :- Classes are blueprints of an Object. A class can have many Objects.

Object :- An Object is an instance of a class. Objects are everywhere in JavaScript
